Bundle

Within CodaLab the basic unit of self-describing storage is referred to as a bundle. A bundle is an immutable directory that can be zipped up into a file that contains a metadata file that describes the contents of the bundle. The specification for bundles is available in the Github repository. There are currently data bundles, algorithm bundles, run bundles, and competition bundles.

Competition

An organized way for users to upload either a program or a bundle with predictions. An evaluation program will automatically score these predictions against the dataset provided by the competition organizer.

Run

A type of Bundle which is equipped with a command-line string that specifies how to execute the code in the Bundle. Execution produces an output directory and various statistics.

User

Someone who creates programs, datasets, worksheets, and participates in competitions.

Worksheet

Worksheets combine code execution, text, mathematics, plots and rich media into a single document, facilitating collaboration and experimentation.
